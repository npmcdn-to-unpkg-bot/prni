  │   ├─Abstract……………………………………………………………………………………… This record holds the multibeam bathymetry chart of Kolonia in Pohnpei Island.  Work was carried out over ¶ a 14-day period in June/July 2006 and involved multibeam bathymetric mapping, single-channel seismic profiling, current metering and CTD (conductivity temperature depth) profiling. ¶ Detailed bathymetric mapping was completed in the following areas of the State of Pohnpei: 1) Kolonia Harbour and Lagoon; 2) Sokehs Passage and Lagoon; 3) Pohnpei Passage; 4) Matalanim Harbour; 5) Nan Madol Lagoon; 6) Rontiki Harbour; and 7) offshore bathymetry of Pohnpei Island. ¶ The offshore bathymetry chart of Pohnpei is available for download from the distribution section of this metadata record with appropriate permission from the Secretariat. Resultant datasets in XYZ format and related reports are available as a separate metadata record. ¶ This work was initiated by the SOPAC/EU project :Reducing Vulnerability of Pacific ACP States.
  │   ├─Purpose………………………………………………………………………………………… To investigate the seabed and provide information about water depths around the islands using a multibeam echosounder (MBES).This bathymetry datasets can serve as the basic tool for scientific, engineering, marine geophysical and environmental studies, as well as marine and coastal resource management and data from the survey will be used to support activities in fisheries, mineral exploration, coastal management, and geo-hazard studies.

  │       └─Statement………………………………………………………………………… The bathymetric data provided in this report has undergone preliminary processing in that noisy data has been removed.  All positions are referenced to the WGS 84 Datum. Grid co-ordinates are referenced to UTM zone 57N. ¶ Offshore data set file is Pohnpei-offshore.xyz. This file is the processed 8160 and 8101 data collected offshore Pohnpei island. Final grid extracted from processed data files was generated from pohnpei-offhore-processed.mtx with a 25m grid setting with cell centre nearest depth value. The file Pohnpei-offshore.xyz was further cleaned for spikes etc using the terravista.  Data reduced to Tide gauge zero for NTF guage in Pohnpei . ¶  ¶ Pohnpei Harbour. Data set file for Pohnpei harbour is Pohnpei-harbour.xyz. Data set is 5m grid extracted from Hypack project using matrix file Pohnpei-harbour.mtx. ¶  ¶ Matalanim Harbour data set. File for data set is matalanim.xyz. Data set is based on 5m grid cell extracted from processed H2s files using Matalanim.mtx. ¶  ¶ Rontiki harbour data set. Bathymetric file is Rontiki-harb.xyz. Data set based on 5m grid extracted using rontiki-harbour.mtx.
