  │   ├─Abstract……………………………………………………………………………………… A SOPAC bathymetry survey conducted in 2002 using a multi beam swath mapping sounder, which mapped throughout the lagoon and offshore of Manahiki .DGPS was used for navigational control and using Hypack surveying software for planning and logging of the sea floor. .Approximately 200 pearl farms/ boundaries was surveyed using a differential Global Positioning System (GPS) of their location and boundaries and individual data of farms tabulated and linked to Map Info using Microsoft Access. Bathymetry data created with a multibeam mapper was also used as reference in producing a rectified image of Manahiki
