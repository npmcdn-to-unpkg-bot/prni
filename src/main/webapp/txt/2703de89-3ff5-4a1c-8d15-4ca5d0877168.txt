  │   ├─Abstract……………………………………………………………………………………… This record holds the multibeam bathymetry datasets in Tab delimited format (XYZ) of Pohnpei in the Federated States of Micronesia as part of the EU-SOPAC (EDF8/9) mapping programme. This field survey study was completed for the State of Pohnpei, inshore and offshore. Work was carried out over ¶ a 14-day period in June/July 2006 and involved multibeam bathymetric mapping, single-channel ¶ seismic profiling, current metering and CTD (conductivity temperature depth) profiling. ¶ Detailed bathymetric mapping was completed in the following areas of the State of Pohnpei ¶ 1) Kolonia Harbour and Lagoon; ¶ 2) Sokehs Passage and Lagoon; ¶ 3) Pohnpei Passage; ¶ 4) Matalanim Harbour; ¶ 5) Nan Madol Lagoon; ¶ 6) Rontiki Harbour; and ¶ 7) offshore bathymetry of Pohnpei Island. ¶ All resultant datasets are available digitally from here and related reports are available as a separate metadata entry. ¶  ¶ This record holds the underlying bathymetric grid files of seabed around the island of Pohnpei in the Federated States of Micronesia (FSM). Datasets are available in 10m grid, 30m grid and 5m grid. Fieldwork was undertaken over a period of 5 weeks from 13 November 2003 to 7 December 2003 by the Applied Geoscience and Technology Division (SOPAC) of SPC. Survey conducted in Pohnpei achieved part of the 100% coverage of the seafloor from approximately 50 to 100 m depth nearshore area to an average offshore distance of 8 km, reaching maximum water depths of some 1200 m for the FSM survey. Bathymetry datasets is available as XYZ text files and can be downloaded from the appropriate link under Distribution Info with appropriate permission from the Secretariat. This work was initiated by the SOPAC/EU project ‘Reducing Vulnerability of Pacific ACP States.

  │       └─Statement………………………………………………………………………… The bathymetric data provided in this report has undergone preliminary processing in that noisy data has been removed.  All positions are referenced to the WGS 84 Datum. Grid co-ordinates are referenced to UTM zone 57N. ¶ Offshore data set file is Pohnpei-offshore.xyz. This file is the processed 8160 and 8101 data collected offshore Pohnpei island. Final grid extracted from processed data files was generated from pohnpei-offhore-processed.mtx with a 25m grid setting with cell centre nearest depth value. The file Pohnpei-offshore.xyz was further cleaned for spikes etc using the terravista.  Data reduced to Tide gauge zero for NTF guage in Pohnpei. ¶  ¶ Pohnpei Harbour. Data set file for Pohnpei harbour is Pohnpei-harbour.xyz. Data set is 5m grid extracted from Hypack project using matrix file Pohnpei-harbour.mtx. ¶  ¶ Matalanim Harbour data set. File for data set is matalanim.xyz. Data set is based on 5m grid cell extracted from processed H2s files using Matalanim.mtx. ¶  ¶ Rontiki harbour data set. Bathymetric file is Rontiki-harb.xyz. Data set based on 5m grid extracted using rontiki-harbour.mtx.
