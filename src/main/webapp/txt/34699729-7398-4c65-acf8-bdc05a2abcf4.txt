  │       └─Statement………………………………………………………………………… The bathymetric data provided in this report has undergone preliminary processing in that noisy data has been removed.  All positions are referenced to the WGS 84 Datum. Grid co-ordinates are referenced to UTM zone 57N. ¶ Offshore data set file is Pohnpei-offshore.xyz. This file is the processed 8160 and 8101 data collected offshore Pohnpei island. Final grid extracted from processed data files was generated from pohnpei-offhore-processed.mtx with a 25m grid setting with cell centre nearest depth value. The file Pohnpei-offshore.xyz was further cleaned for spikes etc using the terravista.  Data reduced to Tide gauge zero for NTF guage in Pohnpei. ¶  ¶ Pohnpei Harbour. Data set file for Pohnpei harbour is Pohnpei-harbour.xyz. Data set is 5m grid extracted from Hypack project using matrix file Pohnpei-harbour.mtx. ¶  ¶ Matalanim Harbour data set. File for data set is matalanim.xyz. Data set is based on 5m grid cell extracted from processed H2s files using Matalanim.mtx. ¶  ¶ Rontiki harbour data set. Bathymetric file is Rontiki-harb.xyz. Data set based on 5m grid extracted using rontiki-harbour.mtx.
